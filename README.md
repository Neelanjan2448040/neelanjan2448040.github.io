# Neelanjan Dutta | AI & Data Science Portfolio

A premium, interactive portfolio showcasing advanced AI engineering, Data Science projects, and research publications. Built with React, Vite, Framer Motion, and standard modern web technologies.

## 🚀 Key Features

- **Interactive Neural Background**: A live canvas-based neural network animation that responds to user interaction.
- **STAR Method Deep-Dives**: Detailed project explanations using the Situation, Task, Action, and Result framework.
- **System Architecture Flows**: Visual breakdowns of technical architectures for each major project.
- **Modern Tech Stack**: Leveraging React 18, Vite for high-performance builds, and Framer Motion for smooth animations.
- **Responsive Design**: Fully optimized for all devices, from mobile phones to high-resolution monitors.
- **CI/CD Integrated**: Automated build pipelines using GitHub Actions.

## 🛠️ Tech Stack

- **Frontend**: React.js, Framer Motion, Lucide Icons, React Router
- **Styling**: Vanilla CSS with Modern Variables & Glassmorphism
- **Build Tool**: Vite
- **Deployment**: GitHub Pages / Vercel / Netlify
- **DevOps**: GitHub Actions (CI/CD)

## 📁 Project Structure

- `src/components`: Reusable UI components (Hero, Navbar, Timeline, etc.)
- `src/pages`: Main page layouts (Home, Project Details)
- `src/assets`: Image and media assets
- `src/App.css`: Consolidated global styling and design tokens
- `.github/workflows`: CI/CD configuration

## ⚙️ Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Neelanjan2448040/Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
